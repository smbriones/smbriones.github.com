Semantic Menu

Semantics HTML Accessibility

Find a menu and mark it up using semantic HTML.

This means a printed menu. You can use any menu you'd like. Don't worry about CSS this time, there will be plenty of other chances for that.

The focus of this exercise is to strip your markup down to it's purest form. Use HTML elements that give semantic meaning to the content that you're working with.

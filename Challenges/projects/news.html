Semantic News Article

Semantics HTML CSS Typography

Find a newspaper or magazine article, mark it up using semantic HTML, style with CSS.

This means a printed article. You can use any article that you choose, but you'll just want to make sure your markup is the main focus.

Additionally, style the article with CSS, matching the style of the article that you found. This may involve some fun and interesting pseudo CSS selectors and columns and such.

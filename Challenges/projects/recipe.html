Find an online recipe, mark it up with POSH (Plain Old Semantic HTML) and Microdata.

You can find a lot more information on using microdata at Schema.org. Make sure you go check out your results with Google's Rich Snippets testing tool.

“Microdata is a WHATWG HTML specification used to nest semantics within existing content on web pages.[1] Search engines, web crawlers, and browsers can extract and process Microdata from a web page and use it to provide a richer browsing experience for users.” Wikipedia

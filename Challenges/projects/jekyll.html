Jekyll Blog with Github Pages

UX Design Typography CSS

Set up a new blog site using Github Pages and Jekyll for your apprenticeship blog posts.

Try to focus on readability and user experience when designing your blog. Start out with some sketches, create a grid and a modular scale, and find the right typefaces for the job. Don't hesitate to ask for help, though. Jekyll can be tricky.

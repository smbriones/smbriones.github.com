Digital Tools

  • XScope

    I've been using XScope almost every day ever since installing it. It's probably the most useful app on my computer currently. I use it for things like color, alignment, measuring, etc.

  • Fount

    Fount is plug-in that I use a lot. If there's a typeface that I like or want to identify, in just a couple of clicks, I know what it is with Fount.

  • github:help

    Something I have used, but should visit more often is the Github Cheat Sheet.

  • PX to EM conversion

    I use this site all the time for my pixels to em, em to pixel conversion needs.

  • The Shapes of CSS

    I refer to this article almost every time I need to create a CSS shape. It's a great list of basic and some not-so-basic shapes.

  • CSS Lint

    A really cool site that will tell it like it is. Not for the easily offended. It judges your CSS pretty harshly, but will definitely make you more awake of the importance of well-written CSS. I need to remember to use it more myself.

  • ASCII Art Generator

    A great, simple ASCII art generator.

  • CSS Pattern Generator

    I came across Patternify recently and think it's a nice tool to use for creating simple background patterns.

Analog Tools

  • Pencil + Paper.

    I like to sketch out everything with plain paper and a pencil. To add a little more life to my sketches I use:

  • Prismacolor Markers
  • Faber-Castell PITT Artist Pens

Hope this list helps you find a new tool to love.
